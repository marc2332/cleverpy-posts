# Cleverpy Posts

Deployed at: https://cleverpy-posts-marc.vercel.app/

## ✂️ Made using

- TypeScript
- NextJS
- React
- Styled-components
- Feather Icons
- Redux
- Playwright (for e2e testing)
- Eslint
- Prettier

## 🧰 Installing dependencies

```shell
yarn
```

## ⚙️ Running locally

```shell
yarn dev
```

## 🧪 Running tests

```shell
yarn test
```
